Cylinder Model

User-Guide
1. Enter a number of sides (3+).

2. Pick a color using the color picker.

3. Click on the Add Cylinder button.

4. You can use the toggle button to see
    the wireframe model.

5. Use Select Cylinder to select the cylinder
    you want to transform.

6. Play around with the sliders to move,
    scale, and rotate the selected cylinder.
